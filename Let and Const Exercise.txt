-What is the difference between var and let?
 Var can be reassigned and redeclared. Let can be reassiged, but not redeclared
-What is the difference between var and const?
 Var can be reassigned and redeclared. Const can't be reassiged or redeclared
-What is the difference between let and const?
 Let can be reassiged but not redeclared. Const can't be reassiged or redeclared
-What is hoisting?
 It concept variables and functions are moved to the top of the code to be used. 
 In reality they are put into memory during the compile phase.
